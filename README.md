#  Tela de Login Responsiva (HTML + CSS)

Este é um projeto simples desenvolvido com o objetivo de praticar **HTML** e **CSS**, focando na criação de uma **interface de login responsiva**.

##  Sobre o projeto

A proposta foi construir apenas o **front-end** de uma tela de login, sem integração com back-end ou autenticação real.

O foco principal está em:

* Estruturação semântica com HTML.
* Estilização moderna com CSS
* Responsividade para diferentes tamanhos de tela

##  Tecnologias utilizadas

* HTML5
* CSS3

##  Responsividade

A interface foi adaptada para funcionar bem em:

*  Dispositivos móveis
* Notebooks
*  Monitores maiores

Utilizando técnicas como:

* Flexbox
* Unidades relativas (%, vh, vw)
* Media queries

##  Objetivo

Este projeto foi desenvolvido com fins de estudo para:

* Melhorar habilidades em layout
* Treinar design responsivo
* Entender melhor posicionamento e organização de elementos

##  Como visualizar
